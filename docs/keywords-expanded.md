## 📊 Cómo usar esta lista

Yo ya tengo el script SQL que las mete a Supabase automáticamente en el archivo `scripts/bulk-import-keywords-expanded.sql`. Solo lo corres una vez que tengas la DB.

**Distribución mensual sugerida:**
- Mes 1: 30 keywords (15 iniciales + 15 expandidas, foco writing)
- Mes 2: 30 keywords (business + productivity)
- Mes 3: 30 keywords (image + video)
- Mes 4: 30 keywords (coding)
- Mes 5: 30 keywords (hosting + creators)
- Mes 6: 50 keywords (mix + las que queden)

El workflow de n8n `01-keyword-research` agrega 10 más por día automáticamente, así que la cola nunca se acaba.

---

## 🎯 Criterios usados pa' esta lista

1. **Intención comercial clara** — cada keyword es "best", "vs", "review", o "how-to" con ángulo comercial
2. **Diversidad de funnel** — desde awareness ("how to X") hasta decision ("X review")
3. **Productos con afiliados** — 85% de las keywords mencionan productos que tienen programa afiliado
4. **Long-tail** — mayoría 4+ palabras pa' evitar competencia brutal
5. **Año actual (2026)** — incluido en varios titulares pa' señalar contenido fresco
6. **Variedad de dificultad** — mezcla pa' ganar rápido + construir a largo plazo
